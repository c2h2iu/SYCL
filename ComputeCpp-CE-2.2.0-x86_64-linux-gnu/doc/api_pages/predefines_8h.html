COMPUTECPP_DEPRECATED_BY_SYCL_2017(message)

Macro which marks an interface as deprecated due to being deprecated or removed in the SYCL 1.2.1 specification. Redirects to COMPUTECPP_DEPRECATED_API if the chosen version specified by SYCL_LANGUAGE_VERSION is greater than or equal to 2017.
