cl::sycl::multi_ptr< void, asp >::multi_ptr ( void *  ptr)

Initialize the object using the given non address space qualified pointer.

This conversion is defined by the device compiler.

Template Parameters
COMPUTECPP_ENABLE_IFOnly available if the provided pointer is not address space qualified.
Parameters
ptrPointer that is not address space qualified that the class should manipulate
Note
This constructor has to be declared in order for the device compiler to deduce address spaces, but it should not be defined because it should never actually be used.
